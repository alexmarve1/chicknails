Слайдер SWIPER JS.

1. Подлкючаем css и js через CDN на https://swiperjs.com/get-started.
2. Создаем HTML структуру, например:
	<!-- Основной контейнер слайдера -->
	<div class="swiper">
  	<!-- Обязательная оболочка -->
  		<div class="swiper-wrapper">
    			<!-- Слайды -->
    			<div class="swiper-slide">Slide 1</div>
    			<div class="swiper-slide">Slide 2</div>
    			<div class="swiper-slide">Slide 3</div>
    ...
  		</div>
  		<!-- If we need pagination -->
  		<div class="swiper-pagination"></div>

  		<!-- If we need navigation buttons -->
  		<div class="swiper-button-prev"></div>
  		<div class="swiper-button-next"></div>

  		<!-- If we need scrollbar -->
  		<div class="swiper-scrollbar"></div>
	</div>

3. В файле CSS для файла img обязательно прописать max-width: 100%. 
4. Инициализация слайдера:


const swiper = new Swiper('.название слайдера', {
  // Параметра слайдера
  direction: 'vertical', - вертикальный слайд
  loop: true,

  // Если нужна пагинация
  pagination: {
    el: '.swiper-pagination', - отображение пагинации
    clickable: true, - Переключение по точкам пагинации.
    dinamicBullets: - динамические буллеты

    renderBullet: function(index, className(стандартный класс буллета)) {
       return '<span class="' + className + '"> + (index + 1) + </span>';   - Создаем цифру внутри буллета. 
     },	

    type: fraction, - изменение типа на страницы.
    type: progressbar - изменение типа на прогресс 
	
  },

  // Стрелки
  navigation: {
    nextEl: '.swiper-button-next', 
    prevEl: '.swiper-button-prev',
  },

  // Скролл бар.
  scrollbar: {
    el: '.swiper-scrollbar',
  },
});

ВАЖНО!

При создание слайдера внутри флекс элемента - флекс элементу необходимо добавить - min-width: 0;
